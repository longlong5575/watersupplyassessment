# 金额

## 数据来源

- 单站评分来自当前 `资料收集/<镇名>/<镇名>附件资料.docx`。
- 付费基数优先来自当前材料中的金额基础表。
- 如果当前附件包没有金额基础表，使用 `report/assets/common_amount_basis.xlsx`。

## 评分换算

- W >= 90：Ec1 = 1，Ec2 = 1
- 80 <= W < 90：Ec1 = [100 - (90 - W) * 0.5]%，Ec2 = 1
- 70 <= W < 80：Ec1 = [95 - (80 - W)]%，Ec2 = [100 - (80 - W) * 0.5]%
- 60 <= W < 70：Ec1 = [85 - (70 - W) * 1.5]%，Ec2 = [95 - (70 - W)]%

镇级 Ec1/Ec2 为抽检设施算术平均值，报告通常保留三位小数。

## 付费公式

- 可用性付费 = 年可用性付费基数（万元/年）* 10000 / 12 * E1 * Ec2
- 第一至第七批运维服务费 = 月运维基数合计 * Ec1
- 扣减金额 = 基数 - 本次考核计算金额

第八、第九批按当前材料中的运营起算和合同口径处理；缺少新口径时沿用通用基数中的批次数据。
