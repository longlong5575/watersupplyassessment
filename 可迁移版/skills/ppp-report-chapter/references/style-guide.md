# Style Guide: 高仿原文正式报告版

## Overall Feel

Write like an official Chinese performance assessment report. The chapter should read as if it belongs in the original正文, not as an audit memo or assistant explanation.

Prefer:
- “根据农村污水处理设施清单……”
- “本次考核小组分别对……”
- “基本能正常运行，且设施规模与实际处理水量基本相适应。”
- “出水水质均达标排放，但……”
- “根据村民、镇政府代表和考核小组……打分情况……”
- “总体来说，该镇设施安全生产管理基本到位。”
- “赤溪镇农村污水处理设施运行维护绩效评价见下表。”

Avoid final-chapter wording like:
- “我重新计算”
- “附件数据显示” when “根据……资料/评分情况” is more official
- “建议优先整改” unless writing a recommendations chapter
- “复核版/分析版/内部口径”

## Structure

Use this order for a town chapter:

1. Title: `<镇名>农村污水处理设施考核情况`
2. `基本情况`
3. Basic facility paragraph: count, process types, total scale, discharge standard, operation dates.
4. `农村污水处理设施运维考核情况`
5. Sampling paragraph: sampled villages, count above 90, lower-scoring facilities, Ec1/Ec2.
6. `1.产出`
7. `（1）项目运营`
8. `（2）项目维护`
9. `（3）成本效益`
10. `（4）安全保障`
11. `2.效果`
12. `（1）生态影响`
13. `（2）社会影响`
14. `（3）可持续性`
15. `（4）满意度`
16. `3.管理`
17. `（1）组织管理`
18. `（2）财务管理`
19. `（3）制度管理`
20. `（4）档案管理`
21. `（5）信息公开`
22. Evaluation-table lead-in sentence and table.

## Data Use

Use attachment score sheets as the primary source for:
- sampled facilities
- deduction items
- final scores
- Ec1/Ec2
- satisfaction scores

Use project-company materials for:
- facility count, process type, scale, pipeline length, well count
-巡查、维修、保养、污泥、水质、公众调查资料 completeness

Use original正文 only for style comparison or template language unless the user says to replace/modify it.

## Rigor Rules

- Preserve exact numerical conclusions derived from attachments.
- If a source has a clear typo in a proper noun, correct it silently in the formal draft.
- Do not invent missing penalties from photos alone.
- If score sheets are missing, produce a formal descriptive chapter but state outside the chapter that Ec1/Ec2 cannot be finalized.
- Keep payment calculations out of chapter two unless requested.

## Common Phrasing Patterns

Sampling paragraph:

`本次考核小组分别对A（x、y），B（m、n）共N个农村污水处理设施进行绩效评价，K个设施点得分在90分及以上，X、Y设施点得分稍低于90分。具体情况详见附件2<镇名>考核评分表。全镇运维期绩效考核运维服务付费系数Ec1为...，运维期绩效考核可用性服务付费系数Ec2为...。`

Water-quality paragraph:

`该镇抽检的N个农村污水处理设施基本能正常运行，且设施规模与实际处理水量基本相适应。出水水质均达标排放，但有...个设施进水CODCr浓度低于100mg/L，影响设施发挥环保效能；有...个设施进水NH3-N浓度＞28mg/L，超出《PPP项目合同补充协议》9.进水标准中进水水质上限值。`

Management paragraph:

`项目公司提供了运营维护管理制度、生产安全管理制度、财务管理制度等，建立了较为健全的运行维护管理体系和制度。`
