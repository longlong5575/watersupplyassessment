# 规范

## 写作口径

写成正式、偏官方的绩效考核报告，不写内部复核口吻。

优先使用：

- “根据农村污水处理设施清单……”
- “本次考核小组分别对……进行绩效评价……”
- “基本能正常运行，且设施规模与实际处理水量基本相适应。”
- “出水水质均达标排放，但……”
- “总体来说，该镇设施安全生产管理基本到位。”

避免使用：

- “我重新计算”
- “内部复核”
- “技能生成”
- “附件数据显示”这类不正式表达

## 章节结构

单镇第二章按以下节奏写：

1. `<镇名>农村污水处理设施考核情况`
2. 基本情况
3. 农村污水处理设施运维考核情况
4. 产出
5. 效果
6. 管理
7. 绩效评价表

单镇第四章只写本镇问题；多镇或总报告要写共性问题分析。
