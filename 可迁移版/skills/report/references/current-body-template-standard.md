# 当前正文底稿生成规范

`assets/正文底稿.docx` 是已清空旧镇街数据的正式报告底稿。

后续生成报告时，用户通常只提供目标镇街的材料，不再提供新的正文。应直接复制该底稿，保留官方套话、章节框架、表格表头、公式、批注式说明和格式，再把目标镇街的考核事实、表格数据、金额计算、问题描述和附件写入。

底稿中不得恢复赤溪镇、白沙镇、大江镇或其他历史镇街的旧数据。所有镇街名称、村点名称、得分、水质、扣款、整改问题、附件内容均以当次用户提供材料为准。

检查方式以结构核验为主：核对标题、目录、章节、表格表头、目标镇街行、金额公式和附件位置。完整 LibreOffice 渲染只在用户要求或疑似排版损坏时执行。
