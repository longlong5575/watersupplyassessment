# 管理员账号管理说明

- 使用 `管理员账号管理.vbs` 打开独立账号管理入口。
- 管理员账号可以新增、重置、启用或停用全部账号；账号名创建后不能修改。
- 新增或重置时，系统自动生成包含字母和数字的随机 8 位密码。页面只显示一次，请立即交给对应使用人员。
- 平台端和移动端共用数据库中的同一套账号。普通用户只能在平台端左下角点击自己的用户名，修改自己的密码。
- 本地首次初始化默认管理员账号为 `admin`，初始密码由后端 `.env` 中的 `ADMIN_INITIAL_PASSWORD` 决定；正式部署必须按照 `backend/.env.production.example` 设置非默认初始密码和 `SECRET_KEY`。
