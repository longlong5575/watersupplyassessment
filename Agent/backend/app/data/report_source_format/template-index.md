  - 郁南县整县生活污水处理设施捆绑PPP项目
  - 绩效考核
  - 2025年第二季度镇级及上半年度农村设施考核报告
  - 考核单位：广东省建筑设计研究院集团股份有限公司
  - 项目名称：郁南县整县生活污水处理设施捆绑PPP项目绩效考核（2025年第二季度镇级及上半年度农村设施考核报告）
  - 现场考核负责人： 陶艺婷 林敏仪
  - 我方受郁南县住房和城乡建设局委托，依据《郁南县整县生活污水处理捆绑PPP项目合同》（以下简称“《PPP项目合同》”）及补充协议（一）约定和有关法律、法规、标准、规范等文件要求，基于项目公司郁南康达亮科环境治理有限公司提供的有关考核所需资料，对郁南县整县生活污水处理捆绑PPP项目所含的镇级污水处理子项目开展绩效考核工作。
  - 一、考核工作开展情况
  - 根据镇级污水处理厂及污水收集管网建设完成情况，以及郁南县住房和城乡建设局工作安排及合同相关约定，2025年6月23日-26日，由郁南县住房和城乡建设局、广东省建筑设计研究院集团股份有限公司、广东华蓝检测技术有限公司、镇污水处理行政主管单位等各派代表组成考核小组，对桂圩镇、罗顺片区、建城镇、宝珠镇、通门镇、千官镇、大方镇、河口镇、宋桂镇、东坝镇、历洞镇和平台镇镇级污水处理厂及配套管网，以及南江口镇、大湾镇、连滩镇、都城镇镇区污水收集管网进行了2025年第2季度现场考核，对15个乡镇共6059个农村污水处理设施进行了2025年上半年度现场考核，并让各考核组成员及被考核单位现场签字确认。在检查农村污水处理设施时同步开展问卷调查。最后，结合水质检测结果，查阅项目公司和实施机构提供的考核资料，对镇级污水处理厂、镇区污水收集管网、农村污水处理设施进行考核评分。
  - 二、考核评分情况
  - 郁南县12个镇级污水处理厂中，建城镇、千官镇、东坝镇得分在80~90分之间，其余各镇得分均在60-80分之间得分均在90分以上。16个镇的污水收集管网中，都城镇镇区污水收集管网考核评分在90分以上，建城镇、罗顺片区、宝珠镇、通门镇、千官镇、大湾镇、大方镇、河口镇、宋桂镇、东坝镇、历洞镇、南江口镇、连滩镇、平台镇镇区污水收集管网考核评分在80~90分之间；桂圩镇镇区污水收集管网考核评分在70~80分之间建城镇、罗顺片区、通门镇、千官镇、大湾镇、河口镇、宋桂镇、东坝镇、都城镇镇区污水收集管网考核评分在80~90分之间；其余镇区污水收集管网考核评分均在70~80分之间。
  - xx15个乡镇共xx59个农村污水处理设施点中，有52个设施现场运维绩效考核评分在90分及以上，7个设施在80~90分之间。有xx个设施现场运维绩效考核评分在80分及以上，xx个设施在70~80分之间，xx个设施低于70分。
  - 三、发现的主要问题
  - 二是项目公司尚未足额配备运维管理人员，运行管理不规范或不符合《PPP项目合同》约定。如设施巡检频次不足、巡检内容不全，生产运行、维护维修等运维记录不全或不规范，未实现污泥规范化管理，未实现数字化管理。关键岗位未持证上岗，未配备岗位人员必要的安全保护措施等。
  - 三是项目公司整改力度严重不足。如管道堵塞，厂区杂物堆放等问题仍普遍存在。
  - 四是污水处理厂处理水量、进水水质COD浓度、出水水质达标率均偏低。本季度绩效考核各污水处理厂分别采样3次，仅大湾镇建城镇、桂圩镇、千官镇、大湾镇、南江口镇、都城镇平均进水CODCr浓度达到合同约定要求，但波动较大，其余各镇均未达到合同约定要求。第二季度抽检的各个镇级污水处理厂出水水质均达标。根据项目公司提供的污水处理量记录表，除建城镇污水处理厂负荷率达59.5%、宝珠镇污水处理厂负荷率达66.8%、千官镇污水处理厂负荷率达69.5%、东坝镇污水处理厂负荷率达59.2%、平台镇污水处理厂负荷率达55.5%之外，其余各镇均低于50%，其中大方镇、历洞镇污水处理厂负荷率更是低于20%宋桂镇污水处理厂负荷率达51.6%之外，其余各镇均低于50%，其中大方镇、罗顺片区、历洞镇污水处理厂负荷率更是低于20%。本季度考核对11个污水处理厂出水各抽检3次，桂圩镇、宝珠镇、通门镇、河口镇、宋桂镇、平台镇污水处理厂均出现出水不达标的情况，超标指标主要是氨氮、TP。
  - 二是项目公司应足额配备运维管理人员，加强培训力度，提升运维管理水平，有效落实巡检巡查工作，规范设施、管网运行与维护管理和污泥处理处置，有效落实安全管理措施。镇级污水处理厂应尽快设置在线监测装置，加快中控平台和数字化管理建设。
  - 三是项目公司应重视问题整改，开展公众调查，针对日常巡检、考核发现问题以及群众反映问题，建立问题台账，专人跟进，逐个销号，有效落实问题整改。
  - 四是针对设施负荷率偏低，进水COD浓度偏低，出水不达标的情况，建议项目公司着力研究污水处理系统提质增效对策，系统性地提升污水处理设施的收集和处理效能，在逐步完善污水收集管网空白区、消除管网缺陷、恢复管道原有功能，提高污水处理厂进水水量和进水COD浓度的同时，优化一体化处理设施的运行参数，合理设置污泥负荷、污泥龄及污泥浓度等重要参数，合理设置曝气时间，提高污水处理设施的处理效能，保证出水达标排放。
